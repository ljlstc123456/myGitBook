# 字体包括继承bootstrap和自定义的

```
.text-left (居左)
.text-right (居右) 
.text-center (居中) 
.text-justify (两端对齐) 
.text-nowrap (禁止文字自动换行)

.text-lowercase (变为全部小写) 
.text-uppercase (变为全部大写) 
.text-capitalize (变为首字母大写)
```
